# ProofMoney Sample Proof Fixtures

This directory contains sample JSON fixtures for developers and reviewers.

These files are:

- sample data only;
- not public network state;
- not PRM with monetary value;
- not claim data;
- not wallet data;
- not private keys;
- not secrets.

## Safety

Fixtures are for documentation, testing, UI prototyping, and review only.

They must not be presented as real balances, real transactions, or public network data.

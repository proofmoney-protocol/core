# ProofMoney Local Proof Explorer

This is a static local proof viewer for ProofMoney MVP data.

Open:

```text
index.html
```

The explorer expects exported JSON files under:

```text
data/
```

Generate data with:

```bash
proofmoney prepare-explorer
```

## Safety

This is local MVP proof data only.

It is not a public network, token sale, exchange, airdrop, yield product, or production wallet.

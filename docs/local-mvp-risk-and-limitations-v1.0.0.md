# ProofMoney Local MVP Risk and Limitations v1.0.0

## Purpose

This document summarizes the main risks and limitations of ProofMoney Core v1.0.0 local MVP baseline.

---

## Core Limitation

ProofMoney Core v1.0.0 is local MVP software.

It is not a public network.

It is not production financial infrastructure.

---

## No Public Network

There is currently no ProofMoney public network.

There is no public consensus.

There is no public node network.

There is no peer-to-peer network.

There is no mempool.

---

## No Monetary Value

Test units, if any, have no monetary value.

They are not exchange-listed.

They are not redeemable.

They do not represent investment rights.

They do not guarantee future allocation.

---

## No Token Sale

ProofMoney does not offer, sell, or solicit the purchase of PRM.

There is no presale.

There is no private sale.

There is no public sale.

---

## No Airdrop

There is no airdrop claim.

There is no claim page.

There is no eligibility list.

There is no test-to-main conversion promise.

---

## No Yield

ProofMoney does not offer yield, staking reward, interest, guaranteed return, or profit opportunity.

---

## No Exchange Integration

There is no exchange listing.

There is no trading interface.

There is no liquidity guarantee.

---

## No Production Wallet

The MVP wallet is experimental.

It must not be used with valuable assets.

Local key material may be exposed if files are mishandled.

---

## No External Audit

The project has founder-led internal review and hardening documents.

It has not completed an external security audit.

---

## Local Ledger Mutation Risk

The local ledger is stored in local files.

Local files can be manually modified by the user.

Tamper detection can detect obvious inconsistencies but is not production forensic security.

---

## Proof Export Staleness

Proof exports are local snapshots.

They can become stale after local ledger changes.

Regenerate exports after state changes.

---

## Regulatory Uncertainty

Regulatory treatment is uncertain.

Nothing in this project should be interpreted as legal, financial, tax, or investment advice.

---

## Experimental Software Risk

ProofMoney may fail.

The software may contain bugs.

Specifications may change.

The local MVP may not evolve into a public network.

---

## Safety Statement

ProofMoney Core v1.0.0 is a stable local MVP baseline only.

It is not an external audit, public network, token sale, yield product, airdrop, exchange integration, or production wallet.

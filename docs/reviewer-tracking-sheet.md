- investment;
- token launch;
- airdrop opportunity;
- yield opportunity;
- exchange listing;
- early access to coins;
- public network launch.

- local Rust MVP;
- external review;
- open protocol experiment;
- verifiable money integrity;
- developer feedback welcome.

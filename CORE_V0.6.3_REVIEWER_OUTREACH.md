# ProofMoney Core v0.6.3 Reviewer Outreach Pack

## Purpose

This pack supports targeted external reviewer outreach.

It does not change protocol logic.

It does not add monetary functionality.

## Added

```text
docs/reviewer-outreach-playbook.md
docs/outreach-message-templates.md
docs/review-response-log-template.md
docs/first-review-batch-plan.md
docs/reviewer-follow-up-templates.md
```

## Use

Use these documents to:

- contact 5-10 targeted reviewers;
- avoid promotional language;
- handle reviewer questions;
- record findings;
- prepare for `v0.7.0-review-response`.

## Safety Boundary

This pack supports technical review only.

It is not a token launch, public network launch, airdrop, yield product, exchange integration, or fundraising campaign.
